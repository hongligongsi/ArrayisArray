# 定位与设计原则

## 一、定位

### 1. 面向工具类 App（如效率工具、计算工具、格式转换、文件处理、实用小工具等），用于管理员对用户、使用数据、功能开关、广告配置、系统参数等进行统一管理。

### 2. 核心价值

- **统一管理**：将分散的工具功能集中到一个平台
- **数据驱动**：基于用户使用数据进行决策和优化
- **灵活配置**：支持功能开关、广告位、弹窗等远程配置
- **运营效率**：提供数据统计和操作日志，便于问题排查

## 二、设计原则

### 1. 简洁实用，突出数据统计与功能配置

- **信息密度适中**：避免过度拥挤，保证可读性
- **数据可视化**：关键数据通过图表、统计卡片直观展示
- **操作便捷**：常用功能一键触达，减少操作步骤
- **响应式布局**：适配 PC、平板、手机等多种设备

### 2. 权限分级，运营与管理员权限分离

- **角色划分**：
  - 超级管理员：所有权限
  - 管理员：大部分管理权限
  - 运营：数据查看、内容审核、用户管理
  - 客服：工单处理、用户反馈
- **权限控制**：
  - 功能级权限：控制模块访问
  - 数据级权限：控制数据查看范围
  - 操作级权限：控制增删改查权限

### 3. 支持功能开关、广告位、弹窗等远程配置

- **功能开关**：支持一键开启/关闭功能模块
- **广告配置**：
  - 开屏广告：启动时展示
  - Banner 广告：页面顶部横幅
  - 激励视频广告：任务完成时展示
- **弹窗管理**：支持配置弹窗内容、展示时机、跳转链接
- **配置优先级**：支持多级配置，覆盖默认设置

### 4. 操作日志完整，便于问题排查

- **日志类型**：
  - 用户行为日志：登录、使用工具、功能访问
  - 管理员操作日志：配置修改、数据操作
  - 系统异常日志：错误、警告、崩溃
  - 接口日志：API 调用记录
- **日志查询**：
  - 按时间范围筛选
  - 按用户/管理员筛选
  - 按操作类型筛选
  - 按日志级别筛选
- **日志导出**：支持导出为 CSV/Excel

## 三、整体布局

### 1. 左侧：功能菜单

- **菜单结构**：
  - 数据概览（控制台）
  - 用户管理
  - 工具管理
  - 会员与订单
  - 广告配置
  - 反馈与问题
  - 系统配置
  - 权限与日志
- **菜单特性**：
  - 支持折叠/展开
  - 图标+文字标识
  - 当前页面高亮
  - 二级菜单支持

### 2. 顶部：系统名称、搜索、消息、管理员信息

- **左侧区域**：
  - 系统名称/Logo
  - 全局搜索框（支持快捷键）
  - 消息通知（带数量徽标）
- **右侧区域**：
  - 管理员信息（头像、昵称）
  - 主题切换（深色/浅色）
  - 退出登录
- **响应式**：
  - PC：完整显示所有元素
  - 平板：简化部分元素
  - 手机：隐藏次要元素，保留核心功能

### 3. 中间：内容展示区

- **布局方式**：
  - 单栏布局：内容区占满宽度
  - 双栏布局：左侧导航+右侧内容
  - 三栏布局：左侧导航+中间内容+右侧辅助信息
- **内容区特性**：
  - 面包屑导航
  - 页面标题
  - 操作工具栏
  - 内容卡片/表格
  - 分页组件

### 4. 底部：版权与版本信息

- **内容**：
  - 版权信息
  - 系统版本号
  - 技术支持链接
  - 备案信息
- **样式**：
  - 简洁设计
  - 固定在页面底部
  - 深色/浅色主题适配

## 四、核心功能模块（工具类专用）

### 1. 数据概览（控制台）

- **今日统计**：
  - 今日启动次数
  - 活跃用户数
  - 新增用户数
  - 总用户量
- **工具使用统计**：
  - 工具使用次数排行
  - 热门工具 TOP 10
  - 平均使用时长
- **系统健康**：
  - 崩溃/异常次数统计
  - 近 7 日用户增长趋势
  - 使用时长趋势
- **图表展示**：
  - 用户增长折线图
  - 工具使用柱状图
  - 系统健康度仪表盘

### 2. 用户管理

- **用户列表**：
  - 用户 ID
  - 昵称
  - 手机号
  - 设备信息
  - 注册时间
  - 会员状态
- **搜索功能**：
  - 用户 ID 搜索
  - 手机号搜索
  - 设备号搜索
- **用户操作**：
  - 拉黑/解封
  - 查看行为日志
  - 会员开通/续费
- **用户详情**：
  - 使用记录：工具使用记录、功能访问记录
  - 付费记录：开通记录、续费记录
  - 反馈记录：用户提交的反馈

### 3. 工具功能管理

- **功能列表**：
  - 工具名称
  - 是否启用
  - 排序
  - 使用次数
- **功能开关**：
  - 一键开启/关闭某个工具
  - 批量操作
  - 开启原因记录
- **工具配置**：
  - 限制次数：每日免费使用次数
  - 会员要求：是否需要会员、会员等级
  - 是否显示广告：开启/关闭广告
- **排序管理**：
  - 调整首页工具展示顺序
  - 支持拖拽排序
  - 保存排序配置

### 4. 会员与订单管理

- **会员套餐管理**：
  - 新增/编辑套餐
  - 价格设置
  - 时长设置
  - 权益设置
- **订单列表**：
  - 订单号
  - 用户
  - 支付金额
  - 支付状态
- **订单操作**：
  - 退款处理
  - 订单导出
  - 订单详情查看
- **会员管理**：
  - 会员到期提醒
  - 自动续费管理
  - 会员权益查看

### 5. 广告配置管理

- **开屏广告**：
  - 启动时展示
  - 展示时长
  - 跳过方式
  - 跳转链接
- **Banner 广告**：
  - 页面顶部横幅
  - 展示频率
  - 跳过时间
  - 跳转链接
- **激励视频广告**：
  - 任务完成时展示
  - 视频链接
  - 展示次数限制
- **广告数据**：
  - 展示量统计
  - 点击量统计
  - 转化率统计

### 6. 反馈与问题管理

- **用户反馈列表**：
  - 反馈类型
  - 内容
  - 联系方式
  - 处理状态
- **反馈操作**：
  - 回复用户
  - 标记已处理
  - 导出反馈
- **常见问题（FAQ）**：
  - 问题分类
  - 问题描述
  - 答案编辑
  - 发布/下架

### 7. 系统配置

- **App 版本管理**：
  - 版本号
  - 更新说明
  - 强制更新开关
- **启动页配置**：
  - 启动页内容
  - 公告配置
- **功能限制**：
  - 每日免费使用次数
  - 游客限制
- **缓存管理**：
  - 清理缓存
  - 系统参数配置

### 8. 权限与日志

- **管理员账号管理**：
  - 账号列表
  - 角色分配
  - 操作日志查看
- **权限配置**：
  - 角色权限配置
  - 功能权限分配
  - 数据权限分配
- **日志查看**：
  - 管理员操作日志
  - 用户行为日志
  - 接口日志查看
  - 日志导出功能

## 五、视觉规范

### 1. 主色

- **主色调**：科技蓝 `#165DFF`
- **辅助色**：
  - 成功绿：`#00B42A`
  - 警告橙：`#FF7D00`
  - 错误红：`#F53F3F`
- **中性色**：
  - 文字主色：`#333333`
  - 文字辅色：`#666666`
  - 边框色：`#E0E0E0`
  - 背景色：`#F5F5F5`

### 2. 简洁清爽，偏效率工具风格

- **设计特点**：
  - 扁平化设计
  - 圆角卡片
  - 柔和阴影
  - 清晰的层次结构
- **间距规范**：
  - 卡片间距：16px
  - 元素间距：8px
  - 边距：24px
- **字体规范**：
  - 主标题：20px，加粗
  - 副标题：16px，加粗
  - 正文：14px，常规
  - 辅助文字：12px，常规
- **图标规范**：
  - 使用 Ant Design Icons
  - 图标大小：16px（常规）、20px（重要）
  - 图标颜色：跟随主色调

### 3. 交互规范

- **按钮样式**：
  - 主按钮：主色调背景，白色文字
  - 次按钮：白色背景，主色调文字
  - 危险按钮：红色背景，白色文字
  - 按钮圆角：6px
- **表单样式**：
  - 输入框：1px 边框，4px 圆角
  - 聚焦状态：主色调边框，2px 阴影
  - 错误状态：红色边框，错误提示
- **加载状态**：
  - 使用 Ant Design Spin 组件
  - 加载文字：浅灰色
  - 遮罩层：半透明白色

### 4. 响应式设计

- **断点设置**：
  - 手机：< 576px
  - 平板：576px - 992px
  - PC：≥ 992px
- **适配策略**：
  - 手机：隐藏侧边栏，使用底部导航
  - 平板：折叠侧边栏，简化布局
  - PC：完整显示所有元素
